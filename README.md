# PyShop 🛒

PyShop is a modern e-commerce web application developed to provide a smooth online shopping experience. The project focuses on creating a responsive and user-friendly shopping platform where users can browse products, view details, add items to cart, and manage purchases efficiently.
This project demonstrates practical implementation of frontend and backend web development concepts including authentication, product management, shopping cart functionality, and responsive UI design.

##  Features

* Product Listing Page
* Product Details View
* Add to Cart Functionality
* Update & Remove Cart Items
* Checkout Workflow
* Responsive User Interface
* Fast and Interactive Experience

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python
* **Framework:** Django
* **Database:** MySQL
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Asishkakarla2004/PyShop.git
```

2. Navigate to the project folder

```bash
cd PyShop
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open in browser

```bash
http://127.0.0.1:5000/
```
